Nano Banana Pro（Gemini 3 Pro Image）は有料プレビューモデルであり無料枠が存在しないため、「無料枠の残量を確認したい」というFAQ項目が矛盾していることに気づき、削除を希望している。また、他のガイド記事にも同様の矛盾や不整合がないか確認したい。

### 作業内容

`app/routes/guides/api-key-setup.tsx` の「よくある問題」セクションから、「無料枠の残量を確認したい」の項目を削除した。このガイドでは「Nano Banana Proは有料プレビューで無料枠では利用できない」と明記しているため、無料枠の残量確認に関するFAQは矛盾していた。

併せて `app/routes/guides/security.tsx` と `app/routes/guides/_layout.tsx` も確認したが、特に問題は見つからなかった。モデル名「Gemini 3 Pro Image (Nano Banana Pro)」の表記もプロジェクト全体で統一されており、整合性は取れている。

### 改善提案

- ガイド記事を書く際は、料金体系の説明と整合性が取れているか確認するチェックリストがあると良い
